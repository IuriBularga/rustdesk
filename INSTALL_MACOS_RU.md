# Установка TradingMD Remote на macOS

Приложение распространяется в виде файла `.dmg`:

- **Apple Silicon** (M1/M2/M3/M4): `tradingmd-remote-<версия>-aarch64.dmg`
- **Intel**: `tradingmd-remote-<версия>-x86_64.dmg`

Какой у вас процессор: меню  → «Об этом Mac» → строка «Чип» (Apple M…) или
«Процессор» (Intel).

## 1. Предупреждение при первом открытии

Приложение подписано ad-hoc подписью и **не нотаризовано** у Apple, поэтому при
первом запуске macOS покажет предупреждение вида:

> «TradingMD Remote» не удаётся открыть, так как разработчик не может быть
> проверен / приложение получено не из App Store.

Это ожидаемо. Файл в этот момент не удаляется — его нужно один раз разрешить.

## 2. Как открыть

Откройте `.dmg` и перетащите **TradingMD Remote** в папку «Программы»
(Applications). Затем — любой из двух способов.

### Способ A. Через системные настройки (рекомендуется)

1. Попробуйте открыть приложение обычным двойным щелчком — появится
   предупреждение, нажмите «Готово»/«Отмена».
2. Откройте  → «Системные настройки» → «Конфиденциальность и безопасность»
   (System Settings → Privacy & Security).
3. Прокрутите вниз до раздела «Безопасность». Там будет строка о том, что
   «TradingMD Remote» был заблокирован. Нажмите кнопку **«Всё равно открыть»**
   (Open Anyway) и подтвердите паролем/Touch ID.

### Способ B. Через Терминал

Откройте «Терминал» (Spotlight → «Терминал») и выполните:

```bash
xattr -cr "/Applications/TradingMD Remote.app"
```

После этого приложение открывается обычным двойным щелчком.

## 3. Выдать разрешения (обязательно)

Без этих разрешений при подключении вы увидите **чёрный экран** и не сможете
управлять компьютером.

Откройте  → «Системные настройки» → «Конфиденциальность и безопасность»
(Privacy & Security) и включите **TradingMD Remote** в разделах:

1. **Запись экрана** (Screen Recording) — иначе передаётся чёрный экран.
2. **Универсальный доступ** (Accessibility) — иначе не работают мышь и
   клавиатура.
3. **Мониторинг ввода** (Input Monitoring) — если система его запросит.

Приложение само предложит открыть нужный раздел настроек при первом запуске.
После включения переключателя macOS попросит **перезапустить** приложение —
согласитесь.

## 4. Как пользоваться

После запуска приложение показывает ваш **ID** и **пароль**. Сообщите их
специалисту поддержки Trading.md — он подключится к вашему компьютеру.
Исходящие подключения в этой сборке отключены: программа работает только на
приём.

## Если что-то не работает

- Приложение не открывается даже после «Open Anyway» — выполните команду из
  способа B и попробуйте снова.
- Чёрный экран у специалиста — проверьте пункт 3 (Запись экрана) и
  перезапустите приложение.
- Не двигается курсор — проверьте пункт 3 (Универсальный доступ).
- Сбросить ранее выданные разрешения:
  `tccutil reset All md.trading.remote`, затем выдать заново.
